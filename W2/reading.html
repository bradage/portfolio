Chapter 2 Reading Notes

toFixed() method rounds a number to a fixed number of decimal places.

const PI = 3.1415926;
PI.toFixed(3) would produce 3.142

toPrecision() method rounds a number to a fixed number of significant figures that is once again returned as a string (and often using exponential notation.)

2.459.toPrecision(2);
Would produce "2.5"

Chapter 3 Reading Notes

deleting in an array is pretty simple, but it will lleave the value as 'undefined'

avengers = ['cap America, 'Fe Man', 'Thor', 'Hulk Smash'];
delete avengers[3]
avengers would now display as: ['cap America, 'Fe Man', 'Thor', undefined]

pop() returns the last item of the array and removes it.

avengers.pop();
This would return 'Thor' and then remove that entry.

shift() returns the first item of the array and removes it.

avengers.shift();
This would return 'cap America' and then remove that entry.

Converting Sets into Arrays is done vea the spread operator.

shoppingSet = new Set().add('Apples').add('Bananas').add('Beans');
shoppingArray = [...shoppingSet]
Would then output ['Apples', 'Bananas', 'Beans']

Chapter 4 Reading Notes

Function expression is done by assigning an anonymous function to a variable (function with no name)

const goodbye = function() {
console.log('goodbye World!');
};

Function expression is done by assigning an named function to a variable

const goodbye = function bye() {
console.log('goodbye World!');
};

though not recommended, you can also create a function via the Function() Constructor.

const hi = new Function('console.log("Hi World!");');

Default parameters can be done by assigning them in the function declaration.

function hello(name='World'){
console.log(`hello ${name}!);
}

Arrow functions havae a number of advantages over other ways of declaring functions:

  • They are much less verbose than normal function declarations.
  • Single parameters don’t need putting into parentheses.
  • The body of the function doesn’t need placing inside a block if it’s only one line.
  • The return keyword isn’t required if the return statement is the only statement in the body of the function.
  • They don’t bind their own value of this to the function (we’ll see why this is a particularly useful property when we cover objects later in the book)

const add = (x,y) => x + y;

const hellp = () => alert('Hello World!');

const tax = (salary) => {
const taxable = salary - 8000;
const lowerRate = 0.25 * taxable;
taxable = taxable - 20000;
const higherRate = 0.4 * taxable;
return lowerRate + higherRate;
}

The filter() method returns a new array that only contains items from the original array that return true when passed to the callback.

const numbers = [ 2, 7, 6, 5, 11, 23, 12 ] numbers.filter(x => x%2 === 0 ); // this returns true if the number is even
this woudl output [ 2, 6, 12 ]
